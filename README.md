📊 E-Commerce Sales Dashboard (Power BI)
📌 Project Overview

This project presents an interactive E-Commerce Sales Dashboard built using Power BI to analyze sales performance across different dimensions such as revenue, quantity, sales channels, managers, and salespersons.

The dashboard provides a clear overview of business performance and helps identify sales trends, top performers, and revenue distribution across channels.

🎯 Objectives

• Monitor overall sales performance

• Analyze monthly sales trends

• Compare revenue across sales channels

• Evaluate manager and salesperson performance

• Track total quantity sold

📊 Dashboard Features
1️⃣ Key Metrics

• Total Revenue: 4.35M

• Total Quantity Sold: 1M

 These KPI cards provide a quick overview of the business performance.

2️⃣ Quantity by Month

 A horizontal bar chart that displays the total quantity sold for each month, helping identify peak sales periods.

3️⃣ Revenue by Sales Channel

This chart compares revenue generated from different channels:

• Retail

• Distributor

• Online

It helps understand which sales channel contributes the most revenue.

4️⃣ Revenue Distribution by Manager

A donut chart showing revenue share between managers:

• Victor Castro

• Gabriel Azevedo

This helps visualize management performance contribution.

5️⃣ Revenue by Salesperson

A bar chart displaying sales generated by individual salespersons:

• Carla Ferreira

• Julio Lima

• Gustavo Gomes

• Leonardo Cardoso

• Felipe Goncalves

Useful for identifying top performers.

6️⃣ Revenue Trend by Month

A line chart showing monthly revenue growth, helping analyze seasonal trends and sales momentum.

7️⃣ Year Filter

A Year slicer allows users to filter the dashboard and analyze data for specific years.

🛠 Tools Used

• Power BI Desktop

• Data Modeling

• Data Visualization

• Interactive Filtering

📈 Insights from the Dashboard

• Retail channel contributes the highest revenue.

• Sales show steady growth across the months.

• Certain salespersons consistently generate higher revenue.

• Management contribution is fairly balanced between managers.

🚀 How to Use

• Open the .pbix file in Power BI Desktop

• Use the Year filter to analyze different time periods

• Interact with charts to explore deeper insights
